Assignment 1 | Part-A

<link> tag has been used to link favicon image and css file with html file
<nav> tag is used to for storing logo(image) and links in form of unordered list.
<div> tag has been used to define different section in the webpage which include - summary, menu and a button to redirect to other page.
menu has been built using <table> tag to display content(coffee name and prices) in table.  
<button> tag has been used to set up sign up button which redirects user to sign up page
sign up page has been built using <form> tag. It asks user to enter email and password and has a sign up button.
